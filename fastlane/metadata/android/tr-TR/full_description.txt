Coffee, cihaz ayarlarını değiştirmek zorunda kalmadan ekranı uyanık tutmanızı sağlar. Çeşitli yerlerde değiştirilebilir:
* A tile in the quick settings, the place that holds the toggles for e.g. Wi-Fi and Bluetooth. Android 7 veya üzeri gerektirir.
* Uygulamanın kendi içinde bir düğme
* Ana ekranınızda bir kısayol. It can be created in the app, if your launcher supports it.

Coffee, yardımcı uygulama olarak ayarlanabilir (Cihaz ayarları => Uygulamalar => Varsayılan uygulamalar => Asistan uygulaması) ve yardım istendiğinde durumunu değiştirir.

If you have any issues with Coffee, please consult the help dialog of the app.
Some OEMs implement an overly aggressive battery saver that makes this app unuseable. This issue has been reported on devices of the following OEMs:
* Poco
* Samsung
* Xiaomi
